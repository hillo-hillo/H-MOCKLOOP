# H-MOCKLOOP
## 项目背景及意义
人工心脏（血泵）的性能及血液相容性评估至关重要，其中体外血液损伤测试是血泵血液相容性检验的重要途经。临床中植入式血泵是在人体的脉动生理条件下运行，因此恒定流工况下的血液损伤测试结果不足以代表血泵的真实血液相容性。人体循环模拟系统可以为血泵提供脉动测试条件，但现有机械式系统体积预充量大，且回路组件繁杂会引入额外的血液损伤，这为脉动工况下的血液损伤测试带来瓶颈。
本项目旨在研发一套低血量、低血液损伤的混合型人体循环模拟系统，实现脉动工况下血泵的血液损伤评估。本系统基于硬件在环概念，采用集总参数模型实现人体血液循环系统的实时仿真，得到人体相关的生理参数；设计软硬件交互接口来复现上述模型计算出的左心室及主动脉压力；最后将血泵流量数据回传至参数模型，参与迭代计算以响应血泵对“人体”的作用；最终实现真实血泵硬件与虚拟人体血液循环系统的实时交互。本文中集总参数模型考虑瓣膜与血流量和压差的相互作用，瓣膜开关程度与瓣膜两侧压差和血流量成非线性关系，实现了瓣膜关闭时主动脉血流的短暂回流以及主动脉压力波中的重博切迹现象。软硬件交互接口包括为血泵提供脉动压力环境的气动装置，组件精简且血液预充量仅为600ml的血泵测试循环回路以及可以部署集总参数模型和控制算法的实时计算控制平台。系统接入血泵后采用甘油混合液进行测试，结果显示系统可以快速跟踪参数模型所计算出的压力，为血泵提供脉动环境；同时参数模型会根据回传的血泵流量数据实时响应，定性评估血泵对人体的影响。
综上所述，本研究设计的混合型人体循环模拟系统体积小、组件精简且具备为血泵提供不同脉动工况的能力，为后续血液损伤测试奠定了基础，最终助力血泵的设计与评估。
## 项目具体内容：
磁悬浮人工心脏（血泵）是一种心脏辅助装置，为心衰患者提供辅助治疗。临床中血泵是在人体的脉动生理条件下运行，因此本项目旨在研发一套混合型人体循环模拟系统，实现脉动工况下磁悬浮人工心脏（血泵）的评估。
## 项目实现思路：
基于硬件在环概念（软硬耦合），采用集总参数模型实现人体血液循环系统的实时仿真，得到人体相关的生理参数；设计软硬件交互接口来复现上述模型计算出的左心室及主动脉压力；将血泵与测试循环回路耦合，实时采集血泵回路的压力、流量信号，以更新集总参数模型迭代参数来响应血泵对“人体”的作用；最终实现真实血泵硬件与虚拟人体血液循环系统的实时交互。
## 项目实现方法：
* 集总参数模型：将人体器官组织特性类比电路结构，并采用matlab设计可以模拟人体心血管循环系统的数值模型；
* 控制板：基于STM32F407芯片控制板，并通过RT-Thread实时操作系统，实现人体循环数值模型实时迭代、压力和流量等信号采集（ADC、RS485）、电磁阀控制以及上位机交互（USB）等多线程任务；
* 控制器：基于simulink搭建控制系统模型，并通过模型预测控制算法（MPC）设计非线性系统控制器；
* 执行机构：通过DAC输出控制电磁阀开度，实现气动装置内部的压力的变化，为血泵提供脉动测试环境；
* 上位机：采用C#搭建的上位机实现系统状态信息的实时显示及参数控制；
